# 🔥 新功能使用指南

## "生成今日24小时总结" 功能

### 如何访问：

1. **在私聊中发送：** `/menu`
2. **点击按钮：** 🔥 生成今日24小时总结
3. **选择群组：** 从列表中选择要总结的群组
4. **获得结果：** 系统生成过去24小时的总结并自动保存

### 功能特点：

✅ **智能时间范围：** 获取过去24小时的消息（不限于当天00:00-23:59）
✅ **自动保存：** 总结文件以当天日期保存（例如：2025-11-03）
✅ **AI驱动：** 使用GPT-5 nano生成高质量中文总结
✅ **即时可用：** 无需等待自动定时任务

### 命令大全：

| 命令 | 功能 | 使用场景 |
|-----|------|----------|
| `/start` | 显示帮助信息 | 首次使用或查看功能介绍 |
| `/help` | 显示帮助信息 | 与/start相同 |
| `/menu` | 打开交互式菜单 | **🔥 推荐使用！包含所有功能** |
| `/myid` | 获取用户ID | 查看自己的Telegram ID |
| `/stats` | 显示群组统计 | 管理员查看消息统计 |
| `/status` | 显示机器人状态 | 管理员检查运行状态 |
| `/summary [参数]` | 生成总结 | 直接命令方式生成总结 |
| `/summary_history` | 查看总结历史 | 管理员查看历史记录 |

### 文件保存规则：

- **文件名格式：** `summary_chat_{群组ID}_{日期}.json`
- **保存路径：** `data/summaries/`
- **示例文件：** `summary_chat_5048705007_20251103.json`

### 自动定时总结：

- **时间：** 每日 00:00（24点）
- **内容：** 生成前一天的总结
- **文件：** 保存为前一天的日期文件

### 手动今日总结：

- **时间：** 随时可用
- **内容：** 过去24小时的消息
- **文件：** 保存为当天的日期文件

## 使用建议：

🎮 **新用户：** 先发送 `/start` 查看功能介绍
🔥 **日常使用：** 使用 `/menu` 访问所有功能
📱 **移动端：** 按钮界面更加友好
💻 **PC端：** 命令和按钮都很方便

## 注意事项：

⚠️ 只有管理员可以使用AI总结功能
⚠️ 需要至少5条消息才能生成总结
⚠️ 总结内容会根据消息长度自动调整格式
